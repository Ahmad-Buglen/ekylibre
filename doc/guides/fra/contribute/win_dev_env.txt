h1. Procédure d'installation Ruby / Rails

1 - installer RailsInstaller 2.1.0 http://railsinstaller.org/
2 - Ajouter ou vérifier la présence des DLL nécessaires au bon fonctionnement dans le repretoire BIN de Ruby (RailsInstaller\Ruby1.9.3\bin)
libiconv2.dll
libiconv-2.dll
libmysql.dll
libxml2-2.dll
libz.dll
zlib1.dll
3 - Paramétrer DevKit comme indiquer https://github.com/oneclick/rubyinstaller/wiki/Development-Kit
installer python 2.7.2 dans C:\python27 et l'ajouter au PATH
3 - télécharger une version d'ekylibre depuis GitHub et dézipper le répertoire de l'application dans le répertoire de votre choix appelé (R)
4 - paramétrer votre base de données et le fichier lié dans (R)\config\database.yml
5 - initialiser votre application , votre base de données et démarrer votre serveur
en ligne de commande
cd (R)
bundle install
rake db:create db:migrate
rails s
