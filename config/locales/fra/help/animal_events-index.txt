====== Soins ======

Les soins peuvent concerner un [[>animals/index|animal]] ou un groupe d'animaux [[>animal_groups/index|groupe d'animaux]]

===== Gestion de la pharmacie =====

Vous pouvez gérer une pharmacie pour votre exploitation avec la listes des [[>drugs/index|médicaments]] utilisés pour vos soins
Cette pharmacie peut être directement liée à votre stock de produit

Le type de compte est défini **à la création**. Vous ne pourrez pas le modifier une fois créé. 
S'il n'a pas servi vous pourrez toujours le supprimer.
