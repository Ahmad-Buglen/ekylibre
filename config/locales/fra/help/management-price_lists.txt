====== Comment gérer les listes de prix ? ======

===== Utilité des listes de prix =====

Les listes de prix vous permettent de **regrouper vos tarifs en catégories**, c'est à dire, pour un produit, de créer un prix différent pour chaque liste. La colonne commentaire vous permet entre autres de décrire à qui est destiné cette liste.

Le tableau récapitulant vos différentes listes de prix vous informe sur le nombre de prix que contient cette liste, sur la monnaie de référence, ...

Pour les listes de prix, il est possible de les :
  * **créer**
  * **afficher** : [[management-price_lists_display|{{theme:buttons/show.png|afficher}}]]
  * **modifier** : [[management-price_lists_update|{{theme:buttons/update.png|modifier}}]]
  * **supprimer** : {{theme:buttons/delete.png|supprimer}}

===== Création automatique =====

  * Certaines listes de prix sont **créées automatiquement**, lors d'un nouvel achat ou d'une nouvelle vente. Par exemple, lors d'un nouvel achat, une liste de prix au nom du nom du fournisseur sera créée, à moins que celle-ci n'existe déjà.
