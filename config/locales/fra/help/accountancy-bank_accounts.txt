====== Comptes bancaires ======

===== Qu'est ce qu'un compte bancaire ? =====
Il s'agit d'un compte ouvert par une banque au nom d'un particulier ou d'un commerce pour permettre à ce dernier de faire diverses opérations financières telles que déposer, retirer ou emprunter de l'argent, faire des chèques... 

Pour renseigner vos informations concernant vos compte bancaires, il vous suffit de vous munir de vos **Relevés d'Identité Bancaire**.

Ils servent à effectuer des rapprochements bancaires (pointage) ainsi que faciliter les remises en banque.

!!! Ne pas confondre les **comptes bancaires** et les **comptes comptables**. Si le premier concerne un dépôt d'argent à la banque, le deuxième ne concerne qu'une partie de la comptabilité de votre société.

===== Gestion d'un compte bancaire =====
Les comptes sont modifiables à volonté et supprimables tant qu'aucun relevés de compte bancaire n'a été pointé pour ce dernier.

Vous pouvez décrire le compte bancaire et notamment la banque qui le gére. Ces informations pourront être affichées sur des documents tels que des remises en banque.

À partir de votre //RIB//, vous pouvez renseigner les éléments du compte bancaire soit en renseignant directement le code IBAN soit en utlisant les champs classique d'un Relévé d'Indentité Bancaire. Dans tous les cas, les numéros de compte seront vérifié et le code IBAN final sera généré. À titre d'exemple, le code IBAN FR76 1351 5280 8051 3729 7872 234 est le résultat du RIB suivant~:
  - Code banque~: 13515
  - Code guichet~: 28080
  - Numéro de compte bancaire~: 51372978722
  - Clé RIB~: 34
