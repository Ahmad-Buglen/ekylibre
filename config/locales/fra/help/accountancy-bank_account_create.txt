====== Comment créer un compte bancaire ======
Vous pouvez décrire le compte banciare et notamment la banque qui le gére. Ces informations pourront être affichées sur des documents tels que des remises en banque.

Dans un deuxième temps, il faut renseigner les éléments du compte bancaire soit en renseignant directement le code IBAN soit en utlisant les champs classique d'un Relévé d'Indentité Bancaire. Dans tous les cas, les numéros de compte seront vérifié et le code IBAN final sera généré.

À titre d'exemple, le code IBAN //FR76 1351 5280 8051 3729 7872 234// est le résultat du RIB suivant~:
  - Code banque~: //13515//
  - Code guichet~: //28080//
  - Numéro de compte bancaire~: //51372978722//
  - Clé RIB~: //34//
