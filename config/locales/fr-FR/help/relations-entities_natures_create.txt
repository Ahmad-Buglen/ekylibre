h1. Comment ajouter un type de fiche ?

h2. Le nom interne et l'abréviation :
La création d'un type de fiche est relative à la toutes les fiches. Le nom interne est le nom que vous désirez attribuer à votre type de fiche. Celui-ci ne sera pas utilisé hors du logiciel, c'est à dire qu'il vous sert uniquement à savoir ce que désigne l'abréviation. C'est celle-ci qui sera utilisée lors de l'élaboration de documents tels que les factures, devis, ... 

h2. Abréviation incluse dans le nom de la fiche :
Le champ "Abréviation incluse dans le nom de la fiche" doit uniquement être coché dans le cas de la création d'un type qui n'est pas une personne physique. En effet, ce champ est uniquement un moyen de vérifier que lors de la création d'une nouvelle fiche de société, le nom de la société n'est pas uniquement renseigné avec la dénomination sociale de l'entreprise mais également avec son statut ( SA, SARL, ...)
