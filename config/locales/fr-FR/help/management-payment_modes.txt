h1. Les types de paiements

Les types de paiement vous servent à définir un *moyen de paiement* et sont utilisés lors des règlements des ventes.

Vous pouvez :
* *Ajouter* un nouveau type de paiement
* *Modifier* un type de paiement :  <<management-payments_update|{{buttons/update.png|modifier}}>>
* *Supprimer* un type de paiement : {{buttons/delete.png|supprimer}}
