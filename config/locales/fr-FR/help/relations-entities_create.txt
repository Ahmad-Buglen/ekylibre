h1. Comment créer une nouvelle fiche ?

Pour créer une nouvelle fiche, il vous suffit de renseigner les différents paramètres qui vous sont demandés. Certains sont obligatoires (encadrés bleux) et d'autres sont facultatifs (encadrés gris). La création ne peut s'effectuer uniquement si tous les champs obligatoires ont été correctement renseignés.
Aux renseignements de base peuvent s'ajouter les différents champs personnalisés qu'il vous est possible de créer.
