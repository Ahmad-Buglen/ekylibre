====== Achats ======

La liste des achats répertorie tous les achats effectués a vos différents fournisseurs.

===== Comment créer un nouvel achat ? =====

Il vous suffit de sélectionner votre fournisseur dans la liste déroulante présentant vos différents fournisseurs et éventuellement d'ajouter une description à l'achat. Si le fournisseur de votre achat n'a pas encore été créé, vous pouvez le faire grâce au lien "Nouveau..." qui vous redirigera vers le formulaire de création d'un nouveau fournisseur. 

===== Création de l'achat ======
La gestion des achats est similaire à la gestion des ventes. En effet, les rôles sont justes inversés.

==== Ajout d'un produit à la commande ====

**Séléctionner un prix permet de sélectionner le produit.**
Il faut sélectionner le prix à commander dans la liste déroulante correspondante. Si le prix n'a pas encore été créé, vous pouvez cliquer sur le lien "nouveau" qui vous redirigera vers le formulaire de création d'un nouveau prix.

Lorsque vous sélectionnez un prix, **le montant HT et la TVA se mettent à jour** automatiquement. Si vous décidez de **modifier** la taxe ou le montant HT, ce **nouveau prix sera enregistré**, associé à votre achat et deviendra le nouveau prix par défaut du produit.

De plus pour les produits gérés en stock dans plusieurs lieux de stockage, il est possible de préciser le lieu ou sera prélevé le produit.
